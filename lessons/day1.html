Dia 1 - Introdução ao Python

O que é Python?

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. É conhecida por sua sintaxe clara e legível, o que a torna uma ótima escolha para iniciantes em programação.

Onde encontrar o Python?

Você pode baixar o Python gratuitamente no site oficial: python.org. Clique no botão "Downloads" e escolha a versão adequada para o seu sistema operacional.

Como abrir o IDLE?

O IDLE (Integrated Development and Learning Environment) é o ambiente de desenvolvimento integrado que vem com a instalação do Python. Para abrir o IDLE:

  1. No Windows, vá para o menu iniciar e procure por "IDLE (Python)".
  2. No macOS, abra o Launcher e procure por "IDLE".
  3. No Linux, abra o terminal e digite "idle".

Como criar um arquivo .py?

Para criar um novo arquivo Python:

  1. Abra o IDLE.
  2. Clique em "File" (Arquivo) no menu superior.
  3. Selecione "New File" (Novo Arquivo).
  4. Uma nova janela de edição será aberta.
  5. Escreva seu código Python nessa janela.
  6. Salve o arquivo com a extensão ".py" (por exemplo, "meu_programa.py").

Dicas para começar

  • Comece com pequenos programas e exemplos simples para se familiarizar com a sintaxe do Python.
  • Use comentários para explicar o que seu código faz. Comentários começam com "#".
  • Pratique regularmente para reforçar seu aprendizado.
  • Não tenha medo de cometer erros. Aprender com os erros é parte do processo.
  • Explore a documentação oficial do Python e outros recursos online para aprender mais.
